## How BuybidHQ Works

BuybidHQ streamlines the wholesale vehicle bidding process into three simple steps:

### 1. Create Bid Request
Enter vehicle information using our VIN decoder or manual entry, add photos, and set condition details.

### 2. Send to Buyers
Select buyers from your network (or buyer groups) and send via SMS instantly.

### 3. Receive Offers
Buyers view details through a mobile-friendly link and submit competitive offers.
