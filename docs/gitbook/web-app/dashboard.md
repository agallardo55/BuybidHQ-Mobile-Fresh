# Dashboard Overview

Your command center for managing bid requests and buyer responses.

---

## Overview

The BuybidHQ dashboard is your central hub for creating, tracking, and managing all vehicle bid requests.

**Access:** Available immediately after login at [app.buybidhq.com](https://app.buybidhq.com)

---

## Dashboard Layout

### Top Navigation Bar

**Left Side:**
- **BuybidHQ Logo** - Click to return to dashboard
- **Dashboard** - Main view (you are here)
- **Buyers** - Manage your buyer network
- **Settings** - Account preferences

**Right Side:**
- **Create Bid Request** - Primary action button (green)
- **Notifications** - Bell icon with unread count
- **Profile Menu** - Account settings, billing, logout

The dashboard shows all your bid requests in a card-based layout:

## Common Questions

**Q: How many bid requests can I see at once?**
A: Default is 25 per page. Change in settings (10, 25, or 50).

**Q: Can I see other team members' bid requests?**
A: Yes, if you're on a team account. All members see all requests.

**Q: What happens to archived requests?**
A: They move to "Archived" folder but can be restored anytime.

**Q: Can I customize the dashboard layout?**
A: Limited customization available (sort, filter, cards per page). More options coming soon.

**Q: Why don't I see some bid requests?**
A: Check your active filters. Click "Clear Filters" to see all requests.

## Troubleshooting

### Dashboard Not Loading

**Problem:** Dashboard is blank or shows loading spinner

**Solutions:**
1. Refresh the page (Ctrl+R or Cmd+R)
2. Clear browser cache and cookies
3. Try incognito/private browsing mode
4. Try different browser
5. Check internet connection
6. Contact support if persistent

---

### Cards Not Displaying Correctly

**Problem:** Bid request cards look broken or cut off

**Solutions:**
1. Zoom browser to 100% (Ctrl+0 or Cmd+0)
2. Update browser to latest version
3. Disable browser extensions temporarily
4. Clear cache and reload
