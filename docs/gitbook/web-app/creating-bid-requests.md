# Creating Bid Requests

Learn how to create and send bid requests to your wholesale buyers.

---

## Overview

Bid requests are the core of BuybidHQ. They allow you to send vehicle information and photos to multiple buyers instantly via SMS.

**Time to Complete:** 3-5 minutes per vehicle
**Available On:** All plans (Beta, Connect, Annual)
**Required Role:** Member or higher

## Step-by-Step Guide

### Step 1: Start a New Bid Request

1. Log in to [app.buybidhq.com](https://app.buybidhq.com)
2. Click **Create Bid Request** button in the top right
3. You'll see the bid request form

---

### Step 2: Enter Vehicle Information

You have two options for entering vehicle details:

{% tabs %}
{% tab title="Option A: VIN Decoder (Recommended)" %}

**Using VIN Decoder:**

1. Locate the VIN field at the top of the form
2. Enter the 17-character VIN
3. Click **Decode VIN** button
4. Vehicle information automatically fills in:
   - Year
   - Make
   - Model
   - Trim
   - Engine
   - Transmission
   - Drivetrain

{% hint style="success" %}
**Pro Tip:** VIN decoding works for 1990+ vehicles and saves significant time!
{% endhint %}

**Supported Vehicles:**
- ‚úÖ All major brands (Honda, Toyota, Ford, BMW, Mercedes, etc.)
- ‚úÖ 1990 or newer vehicles
- ‚ùå Motorcycles, ATVs, UTVs, RVs (not supported)

[Troubleshooting VIN Issues ‚Üí](../troubleshooting/vin-issues.md)

{% endtab %}

{% tab title="Option B: Manual Entry" %}

**Manual Entry:**

1. Click **Manual Entry** (if VIN decode fails or not available)
2. Select from dropdowns:
   - **Year:** Choose model year
   - **Make:** Choose manufacturer
   - **Model:** Choose vehicle model
   - **Trim:** (Optional) Select specific trim

3. Fill in additional specifications:
   - Engine (e.g., "2.0L Turbo I4")
   - Transmission (e.g., "8-Speed Automatic")
   - Drivetrain (AWD, FWD, RWD, 4WD)
   - Body Style (Sedan, SUV, Truck, etc.)

{% hint style="info" %}
**When to Use:** VIN decode failed, older vehicles, or reconstructed titles without VIN data.
{% endhint %}

{% endtab %}
{% endtabs %}

### Step 3: Add Vehicle Photos

High-quality photos are crucial for competitive offers.

**How to Add Photos:**

1. Scroll to the **Photos** section
2. Click **Add Photos** or drag & drop files
3. Upload 5-10 clear photos

**Recommended Photos:**
- ‚úÖ Front exterior
- ‚úÖ Rear exterior
- ‚úÖ Both sides
- ‚úÖ Interior (front seats, rear seats, dashboard)
- ‚úÖ Engine bay (if accessible)
- ‚úÖ Trunk/cargo area
- ‚úÖ Any damage or wear areas
- ‚úÖ Tires and wheels

**Photo Requirements:**
- Max 10MB per photo
- Supported formats: JPG, PNG, HEIC
- Clear, well-lit images
- No watermarks

{% hint style="warning" %}
**Important:** Photos showing damage honestly lead to better offers. Don't hide issues.
{% endhint %}

[Photo Best Practices ‚Üí](../best-practices/photo-tips.md)

---

### Step 4: Set Vehicle Details

**Mileage:**
- Enter current odometer reading
- Automatically formatted (e.g., "125000" ‚Üí "125,000 miles")

**Condition:**
Choose the condition that best describes the vehicle:

- **Excellent** - Like new, minimal wear
- **Good** - Well-maintained, minor wear
- **Fair** - Some wear, may need minor repairs
- **Poor** - Significant wear, needs repairs

**Additional Details (Optional):**
- Vehicle location
- Special notes or features
- Known issues
- Recent maintenance

---

### Step 5: Select Buyers

Choose who receives this bid request:

**Option A: Select Individual Buyers**
1. Scroll to **Select Buyers** section
2. Check boxes next to buyer names
3. Selected buyers show count at top

**Option B: Select Buyer Group** (Faster!)
1. Click **Buyer Groups** tab
2. Select pre-made group (e.g., "Auction Buyers")
3. All buyers in group are selected

{% hint style="success" %}
**Pro Tip:** Create buyer groups to save time! [Learn about buyer groups ‚Üí](managing-buyers.md#organizing-buyer-groups)
{% endhint %}

## What Happens Next?

### Immediately After Sending

**You'll see:**
- Confirmation message
- Delivery status (Sent, Delivered, or Failed)
- Link to view the bid request

**Buyers receive:**
- SMS notification with vehicle summary
- Mobile-friendly link to view full details
- Form to submit their offer

### When Buyers Respond

**You'll be notified via:**
- Dashboard notification
- Email (if enabled)
- SMS (if enabled in settings)

**View responses:**
1. Go to **Dashboard**
2. Click on the bid request
3. See all offers submitted
4. Contact buyer directly to finalize

## Common Questions

**Q: How many buyers can I send to at once?**
A: Unlimited on all plans. Select as many buyers as you want.

**Q: Can I edit a bid request after sending?**
A: No, but you can create a new one with updated information.

**Q: How long does delivery take?**
A: SMS delivery is instant (within seconds). Buyers typically view within 5 minutes.

**Q: What if a buyer's number is wrong?**
A: Update in [Buyer Management](managing-buyers.md), then resend the bid request.

**Q: Can I see who viewed my bid request?**
A: Yes! View delivery and read status in the bid request details.

## Troubleshooting

### VIN Not Decoding
**Problem:** VIN decoder returns error
**Solutions:**
- Verify VIN is 17 characters
- Check for 1990+ vehicle
- Try manual entry as backup

[Full VIN troubleshooting ‚Üí](../troubleshooting/vin-issues.md)

### Photos Won't Upload
**Problem:** Photos failing to upload
**Solutions:**
- Check file size (max 10MB each)
- Ensure stable internet connection
- Try different browser
- Compress large photos

[Full photo troubleshooting ‚Üí](../troubleshooting/photo-issues.md)

### SMS Not Sending
**Problem:** Buyers not receiving SMS
**Solutions:**
- Verify buyer's mobile number
- Check SMS delivery status
- Contact support if persistent

[Full SMS troubleshooting ‚Üí](../troubleshooting/sms-issues.md)
