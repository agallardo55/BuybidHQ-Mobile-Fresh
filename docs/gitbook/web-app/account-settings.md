# Account Settings

Manage your profile, preferences, and security settings.

---

## Overview

Access all account configuration options from one central location.

**Access:** Click **Profile Menu** (top right) ‚Üí **Settings**

---

## Profile Settings

### Personal Information

Update your account details:

**Editable Fields:**
- **Full Name** - Your display name
- **Email Address** - Primary contact email
- **Mobile Number** - Formatted as (XXX) XXX-XXXX
- **Dealership Name** - Your business name

**To Update:**
1. Go to **Settings** ‚Üí **Profile**
2. Click **Edit** next to any field
3. Make changes
4. Click **Save**

{% hint style="warning" %}
**Important:** Changing your email requires verification. Check your inbox for confirmation link.
{% endhint %}

---

### Profile Photo

**Uploading a Photo:**
1. Click **Upload Photo** in Profile section
2. Select image (JPG, PNG, max 5MB)
3. Crop if desired
4. Click **Save**

**Photo Requirements:**
- Square aspect ratio recommended
- Max 5MB file size
- JPG or PNG format
- Clear, professional headshot works best

## Security Settings

### Password Management

**Changing Your Password:**
1. Go to **Settings** ‚Üí **Security**
2. Click **Change Password**
3. Enter current password
4. Enter new password (8+ characters)
5. Confirm new password
6. Click **Update Password**

**Password Requirements:**
- Minimum 8 characters
- At least one uppercase letter
- At least one number
- Cannot be a commonly used password

---

### Multi-Factor Authentication (MFA)

**How MFA Works:**
- First login each day requires SMS code
- Code sent to your mobile number
- Subsequent logins that day don't require MFA
- Resets at midnight

**Updating MFA Mobile Number:**
1. Go to **Settings** ‚Üí **Security**
2. Update your mobile number in Profile section
3. New MFA codes will be sent to updated number

[Full MFA setup guide ‚Üí](../privacy/mfa-setup.md)

---

### Active Sessions

**View Active Logins:**
1. Go to **Settings** ‚Üí **Security** ‚Üí **Sessions**
2. See all active devices/browsers
3. Review login locations and times

**End a Session:**
- Click **Log Out** next to any session
- Use if you logged in on a public computer
- Forces logout on that device

**Log Out All Sessions:**
- Click **Log Out Everywhere**
- Requires re-login on all devices
- Useful if account compromised

## Common Questions

**Q: Can I change my email address?**
A: Yes. Update in Profile settings. You'll need to verify the new email.

**Q: What happens if I forget my password?**
A: Click "Forgot Password" on login page. Reset link sent to your email.

**Q: Can I have multiple users on one account?**
A: Yes, on all plans. Invite team members in Settings ‚Üí Team.

**Q: How do I change from monthly to annual billing?**
A: Go to Settings ‚Üí Billing ‚Üí Change Plan. Annual saves $589/year.

**Q: Can I temporarily pause my subscription?**
A: No pause option. You can cancel and resubscribe later, but you'll lose current pricing.

## Troubleshooting

### Can't Update Profile

**Problem:** Changes don't save when clicking Save

**Solutions:**
1. Check all required fields are filled
2. Ensure email format is valid (name@domain.com)
3. Ensure phone number is 10 digits
4. Try different browser
5. Clear cache and try again

---

### Not Receiving Notifications

**Problem:** No email or SMS notifications arriving

**Solutions:**
1. Check Settings ‚Üí Notifications (ensure enabled)
2. Verify email address is correct
3. Check spam/junk folder
4. Verify mobile number for SMS
5. Test by creating a bid request and responding yourself
