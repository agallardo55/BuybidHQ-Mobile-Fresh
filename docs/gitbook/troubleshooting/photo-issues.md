# Photo Upload Issues

Resolve problems with uploading vehicle photos to bid requests.

---

## Overview

High-quality photos are essential for competitive offers. This guide helps you troubleshoot common photo upload problems.

**Photo Requirements:**
- Max 10MB per photo
- Formats: JPG, PNG, HEIC
- Recommended: 5-10 photos per vehicle
- Clear, well-lit images

---

## Common Upload Problems

### Photos Won't Upload

**Problem:** Upload fails or gets stuck at 0%

**Common Causes:**
1. File size too large (over 10MB)
2. Unsupported file format
3. Slow internet connection
4. Browser issue
5. Too many photos at once

**Solutions:**
1. Check file size - compress if over 10MB
2. Verify file format (JPG, PNG, HEIC only)
3. Test internet speed (speedtest.net)
4. Upload one photo at a time
5. Try different browser
6. Clear browser cache

---

### Upload Stuck at Percentage

**Problem:** Progress bar stops at 23%, 67%, etc.

**Cause:** Usually slow or unstable internet connection

**Solutions:**
1. Wait 2-3 minutes (large files take time)
2. Check WiFi/internet connection
3. Move closer to WiFi router
4. Disconnect from VPN if using one
5. Cancel and retry upload
6. Use wired connection if available

---

### "File Too Large" Error

**Problem:** Error message says file exceeds size limit

**Why:** Photos over 10MB are rejected

**Solutions:**

**Option 1: Compress Photo**
- Use online tool: TinyPNG, CompressJPEG
- Or use Photoshop/GIMP "Export for Web"
- Target size: 2-5MB per photo

**Option 2: Resize Photo**
- Resize to max 1920x1080 pixels
- Use free tool: IrfanView, Paint, Preview (Mac)
- Photos larger than 1920x1080 unnecessary for web

**Option 3: Take Lower Resolution Photo**
- Adjust camera settings before taking photo
- Most phones: Settings ‚Üí Camera ‚Üí Photo Resolution

---

## Upload Performance Issues

### Slow Upload Speeds

**Problem:** Upload takes forever (5+ minutes per photo)

**Causes:**
- Slow internet connection
- Very large file sizes
- Server congestion

**Solutions:**
1. Test internet speed (need 5+ Mbps upload)
2. Compress photos before uploading
3. Use wired connection instead of WiFi
4. Try uploading during off-peak hours (early morning)
5. Close other apps using internet

---

### Upload Times Out

**Problem:** "Upload timeout" or "Request failed" error

**Solutions:**
1. Refresh the page
2. Check internet connection
3. Try smaller file size
4. Upload during better network conditions
5. Try different browser
6. Disable browser extensions temporarily

---

## Photo Quality Issues

### Photos Appear Blurry After Upload

**Problem:** Photo looks clear on device but blurry after upload

**Causes:**
- Over-compression during upload
- Browser downscaling
- Original photo was low resolution

**Solutions:**
1. Upload higher resolution original (not screenshot)
2. Ensure original photo is in focus
3. Don't zoom/crop before uploading (upload full size)
4. Take photos in good lighting
5. Clean camera lens before taking photos

---

### Photos Rotated Wrong Direction

**Problem:** Photo displays sideways or upside down

**Causes:**
- Camera orientation metadata not recognized
- Portrait photos not rotating correctly

**Solutions:**
1. Rotate photo before uploading:
   - Mac: Preview ‚Üí Tools ‚Üí Rotate
   - Windows: Photos app ‚Üí Edit ‚Üí Rotate
   - iPhone: Photos ‚Üí Edit ‚Üí Rotate
2. Re-upload rotated photo
3. Or take photo in correct orientation

---

### Photos Too Dark

**Problem:** Photos are too dark to see vehicle details

**Prevention:**
- Take photos in bright, natural lighting
- Use outdoor lighting when possible
- Avoid dark parking garages
- Enable HDR on phone camera
- Don't shoot directly into sun (backlighting)

**Fix Existing Photos:**
1. Edit photo to increase brightness:
   - iPhone: Photos ‚Üí Edit ‚Üí Light ‚Üí Brightness
   - Android: Gallery ‚Üí Edit ‚Üí Brightness
   - Computer: Use free editor (GIMP, Photos app)
2. Re-upload edited photo

## Browser-Specific Issues

### Chrome

**Common Issues:**
- Upload progress bar stuck
- Files don't appear after selection

**Solutions:**
1. Update Chrome to latest version
2. Clear cache: Settings ‚Üí Privacy ‚Üí Clear Browsing Data
3. Disable extensions temporarily
4. Try Incognito mode

---

### Safari (Mac/iPhone)

**Common Issues:**
- HEIC format issues
- Upload button not responding

**Solutions:**
1. Convert HEIC to JPG before uploading
2. Update Safari/iOS to latest version
3. Try different browser (Chrome)
4. Disable content blockers

---

### Firefox

**Common Issues:**
- Large files timing out
- Drag-and-drop not working

**Solutions:**
1. Update Firefox to latest version
2. Increase timeout: about:config ‚Üí network.http.response.timeout
3. Use "Select Files" button instead of drag-drop
4. Clear cache

---

### Edge

**Common Issues:**
- Upload progress not displaying
- Multiple uploads conflicting

**Solutions:**
1. Update Edge to latest version
2. Clear cache and cookies
3. Reset browser settings
4. Try Chrome if issues persist

---

## Mobile Upload Issues

### iPhone (iOS)

**Common Issues:**
- HEIC format not always accepted
- Photos take long to upload on cellular

**Solutions:**
1. Convert to JPG before uploading:
   - Settings ‚Üí Camera ‚Üí Formats ‚Üí Most Compatible
2. Use WiFi for uploads (faster, more stable)
3. Ensure iOS is updated
4. Clear Safari cache

---

### Android

**Common Issues:**
- Camera folder not appearing in file picker
- Upload stuck on mobile data

**Solutions:**
1. Grant storage permissions to browser
2. Use WiFi for uploads
3. Try different browser (Chrome recommended)
4. Update Android OS and browser

---

## Drag-and-Drop Issues

### Drag-and-Drop Not Working

**Problem:** Can't drag photos to upload area

**Solutions:**
1. Use "Select Files" button instead
2. Update browser to latest version
3. Try different browser
4. Check if dragging from compressed (zipped) folder - extract first

---

### Wrong Files Being Uploaded

**Problem:** Dragged photos but different files uploaded

**Cause:** Browser picked up wrong selection

**Solution:**
1. Use "Select Files" button for precise control
2. Select files one at a time
3. Verify correct files before clicking Upload

---

## Multiple Photo Upload Issues

### Can't Upload More Than X Photos

**Problem:** Upload stops after certain number

**Limits:**
- Recommended: 5-10 photos
- Maximum: 20 photos per bid request

**If hitting limit:**
- System maximum is 20 photos
- Best practice is 8-10 photos
- More photos doesn't always mean better offers

---

### Some Photos Upload, Others Fail

**Problem:** Mixed success when uploading multiple photos

**Solutions:**
1. Upload photos one at a time
2. Check each file size individually
3. Verify each file format
4. Remove failed photos and re-upload separately

## Advanced Troubleshooting

### Clear Browser Cache

**Chrome:**
1. Settings ‚Üí Privacy and Security ‚Üí Clear Browsing Data
2. Select "Cached images and files"
3. Time range: All time
4. Click Clear data

**Safari:**
1. Preferences ‚Üí Privacy ‚Üí Manage Website Data
2. Click Remove All
3. Or: Develop ‚Üí Empty Caches

**Firefox:**
1. Settings ‚Üí Privacy & Security ‚Üí Cookies and Site Data
2. Click Clear Data
3. Check "Cached Web Content"

---

### Disable Browser Extensions

**Extensions that may interfere:**
- Ad blockers (uBlock, AdBlock Plus)
- Privacy extensions (Privacy Badger)
- Security extensions
- Image modifiers

**To Test:**
1. Open Incognito/Private browsing mode (extensions disabled by default)
2. Try upload again
3. If works, disable extensions one by one to find culprit

---

### Check Internet Speed

**Minimum Recommended:**
- Upload speed: 5 Mbps or higher
- Lower speeds will work but be very slow

**Test Speed:**
1. Visit speedtest.net
2. Click "Go"
3. Check upload speed (not download)
4. If under 5 Mbps, consider using better connection

## Photo Editing Tips

### Recommended Compression Tools

**Online (Free):**
- TinyPNG (tinypng.com)
- CompressJPEG (compressjpeg.com)
- Squoosh (squoosh.app)

**Desktop (Free):**
- GIMP (gimp.org) - Windows/Mac/Linux
- Paint.NET (getpaint.net) - Windows
- Preview - Mac (built-in)

**How to Compress:**
1. Upload photo to compression tool
2. Download compressed version
3. Upload to BuybidHQ
4. Quality should remain excellent

---

### Batch Processing Multiple Photos

**If uploading 10+ photos:**

**Mac:**
1. Select all photos
2. Right-click ‚Üí Services ‚Üí Preview
3. Tools ‚Üí Adjust Size ‚Üí Width: 1920px
4. File ‚Üí Export All

**Windows:**
1. Use IrfanView (free)
2. File ‚Üí Batch Conversion/Rename
3. Set output size 1920x1080
4. Batch convert all

## FAQ

**Q: What's the best photo format?**
A: JPG is best. Widely supported, good compression, small file sizes.

**Q: How many photos should I upload?**
A: 8-10 is ideal. Include all angles, interior, engine, any damage.

**Q: Can I upload photos from Google Drive or Dropbox?**
A: Not directly. Download to device first, then upload to BuybidHQ.

**Q: Can I edit photos after uploading?**
A: Not currently. Delete and re-upload edited version.

**Q: Do photos slow down SMS delivery?**
A: No. Photos are not sent via SMS. SMS contains link to photos hosted on our servers.

**Q: Can buyers download my photos?**
A: Yes, buyers can view and download photos. This is expected behavior.
