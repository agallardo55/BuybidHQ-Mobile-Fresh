## Login & Access Issues

### Can't Log In

**Problem:** Login fails with incorrect credentials

**Solutions:**
1. Verify email address is correct
2. Check Caps Lock is off
3. Try "Forgot Password" to reset
4. Clear browser cache and cookies
5. Try different browser

### MFA Code Not Received

**Problem:** SMS code for daily MFA verification not arriving

**Solutions:**
1. Wait 2-3 minutes (carrier delays)
2. Check mobile number is correct in Settings
3. Verify phone has signal/service
4. Try "Resend Code"
5. Check for carrier blocking (contact carrier)

**Temporary Workaround:**
- Email support@buybidhq.com for emergency bypass
- Include account email and phone number

## Dashboard & Navigation

### Dashboard Not Loading

**Problem:** Blank page or infinite loading spinner

**Solutions:**
1. Refresh page (Ctrl+R or Cmd+R)
2. Clear browser cache:
   - Chrome: Ctrl+Shift+Delete (Cmd+Shift+Delete on Mac)
   - Select "Cached images and files"
   - Click "Clear data"
3. Try incognito/private browsing mode
4. Update browser to latest version
5. Try different browser (Chrome recommended)

---

### Missing Bid Requests

**Problem:** Can't see bid requests you know were created

**Solutions:**
1. Check active filters - click "Clear All Filters"
2. Adjust date range to "All Time"
3. Use search to find specific VIN or vehicle
4. Check if another team member archived them
5. Verify you're logged into correct account

## Creating Bid Requests

### VIN Not Decoding

**Problem:** VIN decoder returns error or no results

**Common Causes:**
- VIN is less than 17 characters
- Vehicle is pre-1990
- Invalid VIN (typo in entry)
- Motorcycle, ATV, or RV (not supported)

**Solutions:**
1. Verify VIN is exactly 17 characters
2. Check for typos (O vs 0, I vs 1, S vs 5)
3. Confirm vehicle is 1990 or newer
4. Use manual entry as fallback

[Full VIN troubleshooting ‚Üí](vin-issues.md)

---

### Photos Won't Upload

**Problem:** Photo upload fails or gets stuck

**Solutions:**
1. Check file size (max 10MB per photo)
2. Verify file format (JPG, PNG, HEIC only)
3. Compress large photos before uploading
4. Check internet connection stability
5. Try uploading one photo at a time
6. Try different browser

[Full photo troubleshooting ‚Üí](photo-issues.md)

---

### Can't Select Buyers

**Problem:** Buyer list is empty or buyers won't select

**Solutions:**
1. Verify you've added buyers first (go to Buyers page)
2. Check if buyers are marked as "Active"
3. Clear search filter in buyer selection
4. Refresh the page
5. Try creating buyer group first, then select group

## SMS & Delivery Issues

### SMS Not Sending

**Problem:** Buyers report not receiving bid requests

**Solutions:**
1. Check delivery status in bid request details
2. Verify buyer's mobile number is correct
3. Ensure number is formatted correctly: (XXX) XXX-XXXX
4. Ask buyer to check spam/blocked numbers
5. Confirm buyer has SMS-capable phone (not landline)
6. Try sending test request to yourself first

[Full SMS troubleshooting ‚Üí](sms-issues.md)

---

### "Delivery Failed" Status

**Problem:** SMS shows "Failed" status

**Common Reasons:**
- Invalid phone number
- Number disconnected
- Carrier blocking
- Landline instead of mobile

**Solutions:**
1. Update buyer's phone number
2. Call buyer to verify correct number
3. Try sending to different buyer
4. Contact support if issue persists

## Buyer Responses

### Not Receiving Response Notifications

**Problem:** Buyers respond but you don't get notified

**Solutions:**
1. Go to **Settings** ‚Üí **Notifications**
2. Ensure email/SMS notifications are enabled
3. Check spam/junk folder for emails
4. Verify email address is correct
5. Verify mobile number for SMS alerts
6. Check notification preferences in account

---

### Can't View Buyer Offers

**Problem:** Dashboard shows responses but can't see them

**Solutions:**
1. Refresh the page
2. Click directly on bid request card (not just hover)
3. Clear browser cache
4. Try different browser
5. Check internet connection

## Account & Settings

### Can't Update Profile Information

**Problem:** Changes don't save when clicking Save

**Solutions:**
1. Ensure all required fields are filled
2. Check email format is valid (name@domain.com)
3. Check phone number is 10 digits
4. Remove special characters from name fields
5. Try different browser
6. Clear cache and retry

---

### Payment Method Won't Update

**Problem:** New card won't save in billing settings

**Solutions:**
1. Verify card details are correct
2. Check billing ZIP matches card
3. Ensure card is not expired
4. Try different card
5. Contact your bank (may be blocking)
6. Email support@buybidhq.com for assistance

## Data & Sync Issues

### Changes Not Saving

**Problem:** Edits revert after saving

**Solutions:**
1. Check internet connection
2. Don't navigate away before "Saved" confirmation appears
3. Disable browser auto-fill
4. Clear browser cache

### Duplicate Bid Requests

**Problem:** Same bid request appears multiple times

**Cause:** Usually clicking "Send" multiple times

**Solutions:**
1. Wait for confirmation before clicking again
2. Check all duplicates - delete extras
