## Overview

Can't log in to BuybidHQ? This guide covers common login issues and solutions to get you back to work quickly.

## Common Login Issues

### Incorrect Email or Password

**Problem:** "Invalid email or password" error

**Common Causes:**
1. Typo in email or password
2. Wrong account (multiple emails)
3. Caps Lock is on
4. Password was changed and you forgot
5. Account doesn't exist (not registered yet)

**Solutions:**
1. Double-check email spelling
2. Verify Caps Lock is OFF
3. Try copy/pasting password (avoid typos)
4. Click **Forgot Password** to reset
5. Verify you created an account (check signup confirmation email)

---

### Forgot Password

**Problem:** Can't remember your password

**To Reset:**
1. Go to [app.buybidhq.com](https://app.buybidhq.com)
2. Click **Forgot Password?**
3. Enter your account email address
4. Click **Send Reset Link**
5. Check email for password reset link
6. Click link (valid for 1 hour)
7. Create new password
8. Confirm new password
9. Click **Reset Password**

**Password Requirements:**
- Minimum 8 characters
- At least one uppercase letter
- At least one number
- Cannot be commonly used password

{% hint style="warning" %}
**Important:** Reset link expires in 1 hour. Request new link if expired.
{% endhint %}

---

### Password Reset Email Not Received

**Problem:** Didn't receive password reset email

**Common Causes:**
1. Email in spam/junk folder
2. Email address incorrect
3. Typo in email field
4. Email delivery delay (5-10 minutes)

**Solutions:**
1. Check spam/junk/promotions folders
2. Wait 10 minutes for delivery
3. Verify email address spelling
4. Add noreply@buybidhq.com to contacts
5. Try different email if you have multiple accounts
6. Request reset link again
7. Contact support@buybidhq.com if still not received after 30 minutes

---

## Multi-Factor Authentication (MFA) Issues

### MFA Code Not Received

**Problem:** Daily SMS verification code not arriving

**Common Causes:**
1. Phone number incorrect in account
2. No cell signal
3. Carrier delay
4. Phone turned off
5. Carrier blocking messages

**Solutions:**
1. Wait 2-3 minutes (carrier delays common)
2. Check phone has signal bars
3. Verify phone number in Settings ‚Üí Profile (once logged in)
4. Click **Resend Code**
5. Try calling your phone to verify it's working
6. Contact support@buybidhq.com for emergency access

---

### MFA Code Invalid

**Problem:** "Invalid code" error when entering MFA code

**Common Causes:**
1. Code expired (valid for 10 minutes)
2. Typo in code entry
3. Using old code from previous login attempt

**Solutions:**
1. Request new code (codes expire after 10 minutes)
2. Copy/paste code carefully (avoid typos)
3. Ensure no spaces before/after code
4. Check you're entering 6-digit code, not other SMS

---

### Can't Access Phone for MFA

**Problem:** Phone lost, broken, or no access

**Solutions:**
1. Email support@buybidhq.com immediately
2. Include:
   - Account email address
   - Explanation of situation
   - Verification information (company name, last login, etc.)
3. Support will verify identity and provide emergency access
4. Update mobile number once logged in

## Account Access Issues

### Account Locked

**Problem:** "Account locked" or "Too many login attempts" message

**Why:** Security feature after 5 failed login attempts within 15 minutes

**Solutions:**
1. Wait 15 minutes before trying again
2. Use **Forgot Password** to reset
3. Contact support@buybidhq.com if urgently need access

---

### Account Suspended

**Problem:** "Account suspended" message

**Common Reasons:**
- Payment failure (paid plans)
- Terms of service violation
- Suspicious activity detected
- Account closure requested

**Solutions:**
1. Check email for suspension notice
2. Resolve payment issue if applicable
3. Contact support@buybidhq.com for reinstatement
4. Provide requested verification information

---

### Account Not Found

**Problem:** "No account exists with this email" message

**Possible Reasons:**
1. Using wrong email address
2. Account not created yet
3. Account was deleted
4. Typo in email entry

**Solutions:**
1. Try alternate email addresses
2. Check signup confirmation email
3. Create new account if never signed up
4. Contact support@buybidhq.com if you believe account exists

## Browser and Technical Issues

### Login Page Won't Load

**Problem:** Blank page or error when accessing app.buybidhq.com

**Solutions:**
1. Check internet connection
2. Try different browser
3. Clear browser cache and cookies
4. Disable browser extensions (especially ad blockers)
5. Try incognito/private browsing mode
6. Check if you can access other websites
7. Restart browser
8. Restart computer

---

### Login Button Not Working

**Problem:** Clicking "Log In" does nothing

**Solutions:**
1. Ensure both email and password fields are filled
2. Press Enter key instead of clicking button
3. Clear browser cache
4. Disable browser extensions
5. Try different browser
6. Check browser console for JavaScript errors (F12)

---

### Infinite Redirect Loop

**Problem:** Keeps redirecting back to login page after successful login

**Cause:** Browser cookie/cache issue

**Solutions:**
1. Clear all cookies for buybidhq.com:
   - Chrome: Settings ‚Üí Privacy ‚Üí Site Settings ‚Üí View Permissions and Data
   - Find buybidhq.com and click "Clear data"
2. Clear browser cache completely
3. Try incognito/private browsing mode
4. Try different browser
5. Disable third-party cookie blocking

---

### Session Expired Error

**Problem:** Logged in but then immediately shows "Session expired"

**Causes:**
- Browser not accepting cookies
- Privacy settings too strict
- Third-party cookies blocked

**Solutions:**
1. Enable cookies in browser settings:
   - Chrome: Settings ‚Üí Privacy and Security ‚Üí Cookies ‚Üí "Allow all cookies"
   - Safari: Preferences ‚Üí Privacy ‚Üí Uncheck "Block all cookies"
2. Add buybidhq.com to allowed sites
3. Disable "Enhanced Tracking Protection" for buybidhq.com
4. Try different browser

---

## FAQ

**Q: How often do I need to enter MFA code?**
A: Once per day. First login each day requires SMS code. Subsequent logins that day don't require it.

**Q: Can I disable MFA?**
A: No. MFA is required for all accounts for security.

**Q: Can I use Google/Apple login?**
A: Not yet. Email/password only currently. Social login coming soon.

**Q: I forgot which email I used to sign up. What do I do?**
A: Check your email accounts for confirmation email from BuybidHQ. Or contact support with your company name.

**Q: Can I change my email address?**
A: Yes. Settings ‚Üí Profile ‚Üí Edit Email. Requires verification of new email.

**Q: My account was deleted. Can I recover it?**
A: Maybe. Contact support@buybidhq.com within 30 days of deletion. After 30 days, data is permanently deleted.
